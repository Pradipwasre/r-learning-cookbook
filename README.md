# R-learning-cookbook
Learning R programming from scratch.

---

### Day 1 – Introduction to R Programming

**Topics Covered**
1. Creating and understanding variables  
2. The `paste()` function  
3. Arithmetic operations in R  
4. Working with vectors  
5. Data types  
6. NA vs NaN  
7. Type conversion  

**Learning Outcome**
- Use R as a calculator  
- Handle missing data (`NA`, `NaN`)  
- Understand data types and conversions  
- Combine text and numbers effectively  

---

### Day 2 – Vectors and Matrices

**Topics Covered**
1. Creating and modifying vectors  
2. Indexing and logical filtering  
3. Vector recycling  
4. Sequence generation using `:` and `seq()`  
5. Repetition with `rep()`  
6. Creating matrices and accessing elements  
7. Matrix attributes and operations  
8. Adding rows and columns  

**Learning Outcome**
- Build and manipulate vectors for analysis  
- Understand how recycling works in R  
- Generate numeric patterns using `seq()` and `rep()`  
- Work with matrices for tabular data and computations  
- Apply real-world examples like price adjustments, tabular data setup, and matrix multiplication
